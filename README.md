# Expresso API
- An express app already setup for faster Web API development

## Features
- Interactive CLI
- MVC
- Core setup
- Typescript/JS
- Database Integrated
- API Support
- Errors
- Caching
- Http Responses
- Logs
- Middlewares

## Technologies
- Node 18.13.0

## Setup
- clone 
- npm i
- npm start or npm run dev

## CLI
- create module
- create controller
- create model
- create router
- create service
